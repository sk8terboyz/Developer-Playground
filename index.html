What is the Playground?

This website is where I want to develop fun/random projects. It will be a fully functioning website, but I am going to try to stay away from server-based projects for awhile. Right now I want to work within the capabilities of HTML, CSS, and JavaScript. JQuery and Bootstrap are libraries I am fairly familiar with so I am using these to help learn, and I plan to use NodeJS and VueJS later on.

What can be found in the Playground?

The Playground is the hub for all my web development projects. There will be a range of projects from simple styling pages to full games. The list of projects will most likely become annoyingly long for me in the future so this may not be true forever, but all projects I am currently working on can be found in the navigation Projects dropdown menu. There are 3 sections currently, the Active Projects, Developing Projects, and Explanations. The active projects are projects that are either finished, or to the point where the core functionality will work. The developing projects are projects that may have some parts working, but still need a lot of work. The explanations page is where each of the projects will be explained.

Why are some pages created/accessible with no content?

I created the basic pages I believe will need content eventually. Currently I am still exploring the different styling of pages and haven't found a style/content I want to learn to produce, so some pages will be blank. I plan to make each page a different type of webpage styling except for the navigation. As you can see with this page, all of the content is centered and within a bordered area. This used to be the most common style websites would use a little over a decade ago, and some sites still use it. I wanted to create this homepage in that style so that I can understand why it was so popular and considered the best format of the time.
