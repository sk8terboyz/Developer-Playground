Home

The home page was created in the older style of webpages where all of the content get consolidated in the center of the page surrounded by some sort of border. When I looked at references, it was primarily used for websites that made excessive use of ads in the free space. They typically had enjoyable background images, but I am still trying to figure out what to do with the excess space. For now I have put some banners to make it feel less empty, but maybe I'll put an infinite looped animation to give it a little of activity.

This does not mean all projects labeled as 'Active' will be updated later on, but the minimum requirement for me to move a project into this section is if the core functionality of the project is working as intended.

Branches

This project was a recreation from Mr. Doob where I really got to dive into some of the capabilities of JavaScript. This project is about a year old as the time of writing this, but it is fairly simple. Each time the user clicks within the canvas, branches are formed from the singular point and each branch will continue branching into any direction and go for a maximum lifespan. The 'Switch-Up' button was created to learn how the branches would look and interact with different background colors of the canvas. Similarly the speed slider was created for the user to be able to change the speed of the branches, which would affect the distance traveled within their lifespan. Unfortunately this feature is not working, and I don't have current plans to fix it, but you never know what will happen in the future.

Dice
